# Mobile Automation – SauceLabs Demo App (Appium + Cucumber)

## Descripción

Este proyecto implementa una automatización de pruebas mobile para validar la funcionalidad del **carrito de compras** en la aplicación Android **SauceLabs Demo App**.

La automatización ejecuta pruebas de regresión que permiten detectar comportamientos inesperados en el APK entregado por el equipo de desarrollo. Estas pruebas se ejecutan cada vez que se recibe una nueva versión del APK para validar la estabilidad de la aplicación antes de que pase al equipo de QA para pruebas manuales.

---

## Tecnologías utilizadas

* **Java**
* **Appium**
* **Selenium WebDriver**
* **Cucumber (BDD)**
* **JUnit**
* **Maven**
